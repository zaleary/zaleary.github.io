Takeout Hydrator

Enrich your Google Maps saved places with Places API metadata.

1 Data Tier

Controls which fields are fetched and billed. All tiers are covered by Google's $200/month free credit for typical personal datasets.

2 Google Cloud API Key
Don't have a key?

Go to console.cloud.google.com, create or select a project, then:

  1. Go to APIs & Services → Enable APIs and enable Places API (New)
  2. Go to Credentials → Create Credentials → API key
  3. Copy the key and paste it below

Google provides a $200/month credit — a personal dataset will stay well within free limits. A billing-enabled project is required regardless.

How to restrict your key

Your key is used directly in the browser and will appear in network requests. To limit exposure: in Google Cloud Console, go to Credentials → your key → API restrictions and restrict it to Places API (New). Under Application restrictions you can also limit it to the HTTP referrer zaleary.tv/*.

3 Upload Takeout CSV Files
How to export your saved places
  1. Go to takeout.google.com
  2. Click Deselect all, then check Saved — not Maps or Maps (your places), those are different exports
  3. Export and download the archive
  4. Open it and navigate to Takeout/Saved/
  5. Upload the CSV files below — each saved list (Favorites, Want to go, etc.) is its own file
