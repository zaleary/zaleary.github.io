# zaleary.github.io

This repository is set up as a GitHub Pages + Jekyll blog.

## Publish

1. Commit your changes.
2. Push to the default branch (`main`) with GitHub Desktop.
3. GitHub Pages will build and publish the site automatically.

## Edit content

- Site config: `_config.yml`
- Home page: `index.md`
- About page: `about.md`
- Blog posts: `_posts/*.md`

Create new posts in `_posts` using:

`YYYY-MM-DD-title.md`

with front matter like:

```yaml
---
layout: post
title: "My Post Title"
date: 2026-02-16 10:00:00 -0500
categories: updates
---
```
